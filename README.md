# LINK CAMP 2025 - Official Website Task

This repository contains the design and development of the landing page for **IEEE LINK CAMP 2025**, the flagship leadership event of the IEEE Kerala Section.

## üéØ Project Overview
The goal was to create a responsive, brand-compliant, and event-focused website. Unlike standard event pages, this project reflects the **exclusive nature** of LINK CAMP by featuring a "Selection Process" timeline rather than a generic ticket purchase flow.

### Key Features:
* **IEEE Brand Compliance:** Strictly adheres to IEEE guidelines using official colors (`#00629B`), fonts (Open Sans), and logo placement policies.
* **Dynamic Hero Section:** Features the official event date (Jan 24, 2026) and location (Kochi).
* **Selection Process Timeline:** A custom visual guide explaining the Task ‚Üí Interview ‚Üí Camp flow.
* **Mobile First Design:** Fully responsive navigation and grid layouts.
